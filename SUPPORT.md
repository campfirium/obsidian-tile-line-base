# Support

For TileLineBase support, choose the channel that fits the request:

- Bugs and crashes: open a GitHub Issue.
- Feature requests and workflow ideas: use the Feature requests & ideas discussion category.
- Setup or usage questions: use the Q&A discussion category.
- Broader conversation and examples: use General chat or Show and tell.

GitHub Discussions are preferred for questions and ideas because they are easier for other users to find, compare, and upvote.
